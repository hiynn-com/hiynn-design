## introduction

**海云标准化平台是制订公司的前端框架标准和规范，目的是为了提升前端开发效率节省开发时间**

## usage

```
yarn add hiynn-design
npm install hiynn-design
```

```
import {HdLayout, HdMaster} from 'hiynn-design'
```

## note

文件夹 dist、es 和 lib 是通过 umd、es 和 commonjs 打包的，而 docs 是文档用的文件夹

1. umd -> dist
2. es -> es
3. commonjs -> lib
4. prod -> docs
