## Api :

### HdRanking

| 参数    | 说明                                                  | 类型    | 默认值 | 版本 |
| ------- | ----------------------------------------------------- | ------- | ------ | ---- |
| data    | 数据源                                                | Array[] | -      | -    |
| url     | 传入接口url(同时设置data 和 url 默认显示 url返回的值) | String  | -      | -    |
| showTop | 控制 top 是否显示                                     | Boolean | false  | -    |
| corner  | 控制四个角样式是否显示                                | Boolean | false  | -    |
| style   | 样式调整                                              | Object  | {}     | -    |

### data

| 参数  | 说明  | 类型   | 默认值 | 版本 |
| ----- | ----- | ------ | ------ | ---- |
| name  | name  | String | -      | -    |
| value | value | number | -      | -    |
