## Api :

### HdRanking

| 参数    | 说明                   | 类型    | 默认值 | 版本 |
| ------- | ---------------------- | ------- | ------ | ---- |
| data    | 数据源                 | Array[] | -      | -    |
| showTop | 控制 top 是否显示      | Boolean | false  | -    |
| corner  | 控制四个角样式是否显示 | Boolean | false  | -    |

### data

| 参数  | 说明  | 类型   | 默认值 | 版本 |
| ----- | ----- | ------ | ------ | ---- |
| name  | name  | String | -      | -    |
| value | value | number | -      | -    |
