## Api

| 参数     | 说明                                   | 类型          | 默认值   | 
| :------- | :-------------------------------------| :-----------: | :------: |
| title    | 统计数值标题                           | String        |  -       |
| value    | 统计数值内容                           | String/Number |  -       |
| precision| 设置数值精度                           | Number        |  0       |
| type     | 设置面板排列格式，可选值为inline、block  | String        |  inline  |
| data     | 统计额外显示内容                        | Object        |  -      |
| description | 统计描述                           | -        |  -      |
